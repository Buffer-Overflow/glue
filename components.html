Components

Available Components

Glue has a lot of pre-made components which are used frequently to create games. At the moment you can choose from the following components to mix in to your base objects:

  • Animatable
  • Clickable
  • Draggable
  • Droptarget
  • Fadable
  • Hoverable
  • Movable
  • Rotatable
  • Scalable
  • Visible
  • Collidable
  • Gravitatable

If a game object in your games needs to exibit behaviour that is not available in a Glue component (yet), you can easily create you own custom component you can mix in with your base object.

Creating a custom component

Creating a custom component is a really simple thing to do, but first you must understand how the module pattern works.

Now that you know how modules works lets start creating our own custom component. First we need to decide what our component will do. For this example it will just print to the console the deltaT of the update cycle and we will call it Printable. 

glue.module.create(
    'my/custom/component/printable',
    function () {
        return function (object) {
            object = object || {};
            object.printable = {
                update: function (deltaT) {
                    console.log(deltaT);
                }
            };
            return object;
        };
    }
);

As you can see it's very simple to have a template for custom component. First we return the function that will construct our component and we add as argument the object to which we will add the different properties and methods.

After that we add the property "printable" to our object. This will be accessible from the BaseObject through "this.printable". The printable property will be a literal object that will contain all the methods and public properties.

To use our custom component we first have to retrieve our custom component module using "glue.module.get". Then we can mis in the module in our base object and add our own methods to it using the add method. 

glue.module.get('glue/game', 'my/custom/component/printable', function (Game, Printable) {
    // create our base object,
    // mix in our custom component
    // and add our update function
    var object = BaseObject(Printable).add({
            update: function (deltaT) {
                this.printable.update(deltaT);
            }
        });

    // add our base object to the game
    Game.add(object); 
});

You will now have your own custom component running inside the game loop.
