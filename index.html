Inclusion

To include Glue in your project you can add Glue as a bower dependency. After you did that, you can include Glue in the head of your HTML5 document, like you can see in the example code below. You need to replace [PATH] with the actual path which holds your bower_components folder.

<head>
    <script src="[PATH]/bower_components/glue.min.js"></script>
</head>

//  Set the functions we want to run at core level
Glue.setCore({
    test: function (value) {
        console.log(value);
    }
});
